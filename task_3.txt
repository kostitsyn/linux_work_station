1. Обновляем хранилище пакетов   sudo apt-get update.
2. Монтировка оптического диска  mount mount /dev/sr0 /путь до директории/, для монтирования предварительно надо создать директорию через mkdir.
3. Запуск установки гостевых дополнений путем запуска файла autorun.sh
4. Устанавливаем пакеты   sudo apt-get install gcc make perl.
